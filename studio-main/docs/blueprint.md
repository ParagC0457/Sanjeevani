# **App Name**: HealthWise Companion

## Core Features:

- Personalized Medication Reminders: Provide timely reminders for medications and vaccines based on user-specific health data and schedules.
- Health Awareness Feed with Symptom Checker: Offer a feed of health awareness content with a friendly chatbot tool that checks for symptoms based on user input, providing initial guidance.
- Documentation Recorder: Allow users to record and store health-related documents. Support PDF parsing.
- Report Explanation Chatbot: Utilize the chatbot tool to provide explanations and summaries of uploaded medical reports.
- User Health Profile: Enable users to input and maintain their health information, including height, weight, age, blood sugar levels, blood pressure, and existing conditions.
- Symptom tracking: Enables users to log their symptoms, so that they may be recalled and/or summarized to health practitioners.
- Integration with Gemini API: Integrate the chatbot features seamlessly, by connecting the Nextjs frontend to the Gemini API.

## Style Guidelines:

- Primary color: beige(#F5F5DC) to convey trust and serenity, drawing inspiration from the trustworthiness often associated with healthcare.
- Background color: Very light green ( (#4a6f11) ) provides a clean and calm backdrop.
- Accent color:(#537d13)adds a touch of sophistication and provides contrast to the primary blue, while echoing the primary's feeling of reassurance.
- Body and headline font: 'PT Sans' (sans-serif) combines a modern and warm feel for clear readability and a friendly user experience(#FFFFFF).
- Use clear, modern icons to represent different health metrics and functionalities.
- Employ a clean, intuitive layout with clear sections for easy navigation and accessibility.
- Subtle transitions and animations to enhance user interaction and provide feedback.
